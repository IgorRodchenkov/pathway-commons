Class PathwayCommonsIOHandler

  • java.lang.Object
    • cpath.client.PathwayCommonsIOHandler

  • public class PathwayCommonsIOHandler
extends Object
    Provides querying and fetching relevant OWL model(s) from old Pathway Commons database (http://www.pathwaycommons.org) using the features of Pathway Commons web service API: (http://www.pathwaycommons.org/pc/webservice.do?cmd=help)

      • getPathways

        public List<List<String>> getPathways(String id)
                               throws IOException
        Get pathways to which the entity, specified by its id, is related and returns a two-dimensional ArrayList containing all results as rows and their subproperties as columns. First row contains the column types (Database:ID, Pathway_Name, Pathway_Database_Name, CPATH_ID). Pathway Commons definition: Retrieves all pathways involving a specified physical entity (e.g. protein or small molecule). For example, get all pathways involving BRCA2.
        Parameters:
        id - entity id
        Returns:
        An ArrayList containing "PHYSICAL_ENTITY_ID_NOT_FOUND" as the second column in case of no results
        Throws:
        IOException
