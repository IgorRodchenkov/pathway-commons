      • exportData

        public static void exportData(String output,
              String[] uris,
              boolean outputAbsoluteUris,
              String[] datasources,
              String[] types)
                       throws IOException
        Exports a cpath2 BioPAX sub-model or full model to the specified file.
        Parameters:
        output - - output BioPAX file name (path)
        uris - - optional, the list of valid (existing) URIs to extract a sub-model
        outputAbsoluteUris - - if true, all URIs in the BioPAX elements and properties will be absolute (i.e., no local relative URIs, such as rdf:ID="..." or rdf:resource="#...", will be there in the output file.)
        datasources - filter by data source if 'uris' is not empty
        types - filter by biopax type if 'uris' is not empty
        Throws:
        IOException, - IllegalStateException (in maintenance mode)
