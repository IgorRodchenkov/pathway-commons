Package cpath.query

  • Interface Summary 
    Interface Description
    CPathQuery<T>
    Basic interface to develop specific and convenient query classes to be executed with the cPath2 client.
  • Class Summary 
    Class Description
    CPathGetQuery
    A get by ID/URI query to be executed with CPathClient
    CPathGraphQuery
    A query to be executed with CPathClient
    CPathSearchQuery
    A full-text search query to be executed with CPathClient on the cpath2 server's biopax database.
    CPathTopPathwaysQuery
    A "get top pathways" query to be executed with CPathClient on the cpath2 server's biopax database.
    CPathTraverseQuery
    A property 'traverse' query to be executed with a CPathClient on the cpath2 server's biopax db.
