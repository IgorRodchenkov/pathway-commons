        Map<String,List<Object[]>> downloadsTimeline(LogType logType,
                                           String name)
        Aggregates the history (log) of a particular service category, LogType, provided for users on each day; the result is a list of [date, count] pairs for that category and service name. The result also includes the total counts by date for all names in this category altogether (logType is used for the key).
        Parameters:
        logType - not null
        e.g, - provider's name, command, format, or filename (depends on the log type)
        Returns:
        name (key) - to array of Object[2] map, e.g., if logType eq. COMMANDS, and name were null: 
         {
        "All Commands": [["2013-11-13", 123], ["2013-11-14", 456],..], 
        "search" : [["2013-11-13", 123],..], 
        "get" : [["2013-11-13", 123],..], 
        "traverse" : [["2013-11-13", 123],..],..
 }
        If name, e.g. "search", were provided, the result would be: 
         {
        "All Commands": [["2013-11-13", 123], ["2013-11-14", 456],..], 
        "search" : [["2013-11-13", 23], ["2013-11-14", 45],..]
 }

      • downloadsWorld

        List<Object[]> downloadsWorld(LogType logType,
                            String name)
        The number of client requests served, grouped by country code. The result is a list of [country, count] optionally filtered by the service type and name if the arguments were provided, as follows:
        • neither type nor name - total counts;
        • only name is provided - counts for this name (ignoring type, which usually implies one particular type, because a name cannot belong to different types...);
        • type, except TOTAL, and name -> counts for all events of the name and type (in fact, it does not depend on the type; hence same as above).
        • type, except TOTAL, but no name -> counts for all events of the type;
        Parameters:
        logType - filter by (limit to one) category, e.g., provider, file, command, or total, etc.
        name - filter by name (e.g, provider's name, command, format, or filename)
        Returns:
        array of Object[2] items: 
         {[['Country', 'Downloads'], ['US', 123], ['CA', 456], ['RU', 45],..]}
